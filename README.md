# Developer Portfolio — React + Vite + Tailwind

## ⭐ If you enjoy the project, consider giving the repository a star!

A fast, modern, and fully responsive developer portfolio built with **React**, **Vite**, and **Tailwind CSS**.

### 🔗 Live Demo
[emilisvoropajevas.dev](https://emilisvoropajevas.dev)

The portfolio includes:
- Custom typewriter hook animations
- Interactive terminal interface
- Fully customizable terminal commands and virtual file system
- Smooth animations powered by Motion
- Responsive layout for all screen sizes

---

## ⚙️ Tech Stack

- ⚛️ React
- ⚡ Vite
- 💨 Tailwind CSS
- 🔧 JavaScript (ES6+)
- 🎞️ Motion

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/emilisvoropajevas/portfolio.git
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```

---


## 🛠️ Customization

### Update portfolio content & customize layout and sections

Edit components and pages inside:

```bash
/src/components/
```

### Customize the interactive terminal

Add terminal commands and virtual files inside:

```bash
src/components/Terminal.jsx
```

Update the:

- `commands` object
- `files` object

located near the top of the file.
